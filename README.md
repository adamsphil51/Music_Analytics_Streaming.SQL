# Music_Analytics_Streaming

This project demonstrates **data analytics for the music streaming industry**.  

**Objectives:**  
- Analyze **streaming data by song, artist, and genre**  
- Track **monthly active users and listener engagement**  
- Monitor **revenue per artist/song**  
- Provide insights for **marketing, playlist curation, and artist growth**  

**Skills demonstrated:** SQL, Revenue & Engagement Analytics, Artist & Song Performance, Listener Behavior, Dashboard-Ready Queries
